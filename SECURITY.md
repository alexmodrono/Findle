# Security Policy

If you believe you have found a security vulnerability in Findle, please do not open a public issue.

Instead, email:

amodrono@alu.icai.comillas.edu

Please include:
- a description of the issue
- steps to reproduce it
- the affected version or commit
- any relevant screenshots or logs
- whether you believe the issue affects authentication, tokens, local storage, or File Provider behavior

I will do my best to acknowledge receipt quickly and investigate responsibly.

Please avoid publicly disclosing the issue until it has been reviewed and, if necessary, fixed.
